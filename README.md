```sh

│  ├─190313
│  ├─190320

```
`ppt` : 발표 자료
`src` : 구현한 개별 코드

### 파일 양식
* 발표 자료는 `ppt` 폴더에 `날짜_이름.pptx` 형식으로 저장.
* 구현한 코드는 `src` 폴더에 `날짜_문제_이름.cpp` 형식으로 소스파일을 저장.

### 커밋 양식
* 파일을 추가한 경우 `Add: Commit Message` 형식으로 커밋.
* 파일을 수정한 경우 `Fixed: Commit Message` 형식으로 커밋.
